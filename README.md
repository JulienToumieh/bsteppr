# BSteppr
A simple and intuitive Drum Sequencer made with Godot 4

# Features
- Intutive interface
- Saving and loading loops (supports autosave)
- Tempo & Swing
- Velocity control
- Changing drum kits
- Sequenced & repeated loops
- Cross platform support
